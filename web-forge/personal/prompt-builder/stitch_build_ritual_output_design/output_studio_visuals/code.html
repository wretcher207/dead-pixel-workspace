Composition Rules

Frame the void. This project mandates an editorial embrace of negative space. Every element must be weighed against the silence it disrupts. Avoid symmetry; it is the death of curiosity.

Prioritize off-center focal points and extreme tonal shifts. The composition should feel like a static frame from a 16mm archival reel—unsteady, intentional, and fleeting.

Project Brief Summary

A cinematic exploration of architectural isolation, focusing on the tension between cold structures and the warmth of lingering memories.
