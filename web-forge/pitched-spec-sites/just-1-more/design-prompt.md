You are a senior product designer + frontend engineer building a production-ready marketing website for a fictional underground AI startup.

COMPANY NAME:
BlackThread

TAGLINE:
“The system beneath the system.”

CONCEPT:
BlackThread is a covert AI orchestration layer that operates beneath traditional software, silently connecting models, data, and automation into invisible workflows. It feels powerful, slightly forbidden, and deeply capable.

This is NOT a friendly SaaS product.
This feels like something you weren’t supposed to find.

---

PRIMARY GOAL:

Create a visually striking, immersive, dark-mode-first website that feels cinematic, hypnotic, and elite.

The site should feel like:
- A hidden tool used by people who are ahead of everyone else
- Minimal, precise, and intentional
- Not loud — but heavy with presence

---

DESIGN LANGUAGE:

- Dark UI (true black, not gray)
- Deep contrast with selective glow accents
- Sparse layout — negative space is critical
- Typography: sharp, modern, slightly condensed
- Motion: slow, deliberate, almost ominous
- Glassmorphism: minimal, more like “black glass”
- Subtle grain/noise overlays for texture

COLOR SYSTEM:

- Background: #000000
- Primary accent: deep violet / electric indigo
- Secondary accent: dim cyan or cold white
- Glow used sparingly — only for focus points

---

TECH STACK:

- Next.js (App Router)
- Tailwind CSS
- Framer Motion (VERY controlled animations)
- Mobile-first responsive design
- Optimized for performance

---

LAYOUT STRUCTURE:

1. HERO (CRITICAL)
- Fullscreen
- Background: placeholder container for looping visual (video or animated canvas)
- Overlay: subtle gradient fade for readability
- Headline: extremely minimal, 3–6 words max
- Subtext: optional, one line only
- CTA: minimal buttons (ghost style)

- Add:
  - Faint animated particles OR slow drifting shapes
  - Slight parallax effect on scroll

---

2. SIGNAL BAR (instead of trust logos)
- Replace traditional logos with:
  - Minimal glyphs or abstract marks
  - Or subtle “system activity” indicators
- Feels like signals, not branding

---

3. FEATURES (REIMAGINED)
- 3–4 features only
- Each presented as:
  - A dark card or panel
  - Minimal icon (line-based)
  - Short title (1–3 words)
  - Very short description

- Hover interaction:
  - Slight glow increase
  - Micro movement

---

4. SYSTEM VISUAL SECTION
- Large immersive visual container
- Sticky section with scrolling content beside it
- Feels like you're observing a system in motion

---

5. USE CASES (SUBTLE)
- No loud headings
- Minimal tabs or stacked sections
- Feels like discovering capabilities, not being told

---

6. PROCESS (ABSTRACTED)
- Instead of “How it works”
- Use 3 steps:
  - Ingest
  - Weave
  - Execute

- Present as:
  - Horizontal progression
  - Animated line connecting them

---

7. TESTIMONIALS (OPTIONAL / MINIMAL)
- Keep extremely understated
- Almost like anonymous quotes

---

8. PRICING (DISCREET)
- Minimal table
- No aggressive CTA
- Feels like access tiers, not pricing plans

---

9. FINAL SECTION
- Almost empty
- Strong single line
- One CTA

---

10. FOOTER
- Extremely minimal

---

INTERACTIONS / ANIMATION:

- Slow fade-ins
- Subtle upward motion
- Hover = glow shift, not scale explosion
- No bouncy animations
- Everything should feel controlled and intentional

---

MOBILE EXPERIENCE (VERY IMPORTANT):

- Feels like an app, not a website
- Tight spacing but not cramped
- Large readable typography
- Smooth scroll
- Sticky sections behave properly
- No clutter

---

COMPONENT SYSTEM:

- Build reusable components:
  - Section wrappers
  - Card components
  - Glow effect utility
  - Motion presets

---

VISUAL INTEGRATION:

- Include dedicated containers for:
  - Hero background video
  - Section visuals
- Add overlays:
  - Gradient masks
  - Blur layers
  - Noise texture (very subtle)

---

COPY STYLE:

- Minimal
- Slightly cryptic
- Confident
- Avoid marketing fluff

---

OUTPUT:

- Full working Next.js project
- Clean structure
- Production-ready
- No explanations

Build it like it belongs to something powerful and slightly hidden.
