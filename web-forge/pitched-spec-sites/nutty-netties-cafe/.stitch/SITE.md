# Nutty Netties Cafe — Site Vision

## 1. Overview

A spec website for Nutty Netties, a fictional southern Maine cafe. Built by Dead Pixel Design as a portfolio piece. The site must feel warm, whimsical, storybook-meets-polished-web — not a generic cafe template.

**Tagline:** "A little nutty. Completely worth it."
**Location:** Biddeford, Maine
**Vibe:** Cozy neighborhood cafe with a wink. Handmade, tactile, editorial.

## 2. Brand Identity

- Subtle illustrated squirrel motif recurring throughout (tasteful, not cartoonish)
- "Nettie note" Caveat-font annotations on featured items
- Recipe card / paper label / corkboard aesthetic for interactive elements
- Warm coffeehouse textures: paper grain, linen, ceramic, wood, frosted glass

## 3. Tech Stack

- Next.js App Router + Tailwind CSS + Framer Motion
- Static export (`output: 'export'`)
- Google Fonts: Cormorant Garamond, DM Sans, Caveat
