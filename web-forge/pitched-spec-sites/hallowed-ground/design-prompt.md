# 🧱 SITE STRUCTURE

## 1. HERO SECTION
- Full-screen cinematic background (video-ready)
- Studio name
- Short, powerful tagline
- Primary CTA: "Book Consultation"
- Secondary CTA: "View Work"

## 2. ARTIST / PORTFOLIO SECTION
- Grid layout with filtering (style, artist)
- Lightbox modal for viewing work
- Each artist has:
  - Name
  - Specialty
  - Short bio
  - CTA: "Book with [Artist]"

## 3. ABOUT / BRAND STORY
- Short but compelling narrative
- Focus on craftsmanship, experience, and trust

## 4. BOOKING SYSTEM (AI-ENABLED MOCK)
- Interactive consultation form:
  - Tattoo idea
  - Placement
  - Size
  - Reference uploads
- "AI Assistant" mock feature:
  - Suggests artist based on input
  - Recommends approximate pricing tier
- Calendar UI (mocked, not functional yet)

## 5. ONLINE PAYMENT / DEPOSIT SYSTEM (MOCK)
- Deposit selection UI
- Pricing tiers (small / medium / large pieces)
- Clean checkout UI (non-functional placeholder)

## 6. TESTIMONIALS
- Clean, minimal, high-trust presentation

## 7. FAQ
- Booking process
- Pain expectations
- Aftercare basics

## 8. LOCATION + CONTACT
- Embedded map (Portland, Maine)
- Studio hours
- Contact CTA
