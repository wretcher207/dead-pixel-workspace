## Feed post

Seedance 2.0 + Claude Code Creates $10k Websites in Minutes  
  
This stuff used to cost tens of thousands of dollars and weeks of production time.  
  
Now it takes minutes.  
  
Here's the workflow:  
  
1️⃣ Generate a reference image with Nano Banana2  
  
2️⃣ Feed that image into Seedance as both the first and last frame  
→ This creates a seamless loop that plays endlessly on the site  
→ Use my free prompting skill to write the actual video prompt  
  
3️⃣ Drag the video into Claude Code and say "build me a website"  
→ Use plan mode to align on the feel before it starts building  
→ Install the frontend design skill for better taste  
  
4️⃣ Iterate by dragging in reference designs from Dribbble or Awwwards  
  
5️⃣ Push to GitHub, connect Vercel, and you're live  
  
I just dropped a 23 minute YouTube video walking through the entire process from zero to a deployed site.  
  
The whole point is giving users a feeling when they land on your site. Not just text and stock photos. Actual motion. Actual presence.  
  
Claude Code handles all of it.
