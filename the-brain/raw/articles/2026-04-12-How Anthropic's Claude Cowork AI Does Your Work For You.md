Claude Cowork is here.

Anthropic's Cowork is an [agentic AI assistant](https://aimagazine.com/news/behind-the-rise-of-ai-agents-replacing-human-recruiters) that you can talk to like an AI coworker – to help you plan and do menial tasks for you.

Be it organising messy folders, creating an expense spreadsheet from a scrappy pile of screenshots or compiling together a research report from your notes and scattered sources, Claude Cowork can handle it.

Once the user sets a task for Cowork, it will create a plan and go about solving it – all while keeping the user in the loop on what it’s doing. No hassle, just pure efficiency.

![](https://www.youtube.com/watch?v=UAmKyyZ-b9E)
![Youtube Placeholder](https://i.ytimg.com/vi/UAmKyyZ-b9E/hqdefault.jpg)

By pairing [Claude in Chrome](https://aimagazine.com/news/the-results-of-anthropics-claude-ai-chrome-extension-pilot), users can also outsource tasks that require browser access for Cowork to handle. Talk about an AI tool to boost productivity!

“When we released Claude Code, we expected developers to use it for coding. They did – and then quickly began using it for almost everything else,” Rahul Patil, CTO at Anthropic, says on LinkedIn.

“This prompted us to build Cowork: a simpler way for anyone – not just developers – to work with Claude in the very same way.”

Claude Cowork, currently in research preview, is available for max plan subscribers who can check out the new functionalities through the macOS desktop app.

## How is Claude Cowork different from Claude Code?

Anthropic’s Claude Cowork does everything that Claude Code can, but made with non-coders in mind.

It is more user friendly and approachable for non-coding tasks, though it uses the same foundations as Claude Code.

While Claude Code is CLI (Command Line Interface)friendly, Cowork can be accessed byits user-friendly GUI (Graphical User Interface) designed for those creative users who are not particularly coding inclined.

When [Claude Opus 4.5 hit the market](https://aimagazine.com/news/anthropics-new-claude-opus-4-5-beats-the-best-human-coders) not so long ago, developers rushed to embrace it using Claude Code to streamline coding workflows, while non-coders messed with its [vibe-coding](https://technologymagazine.com/news/vibe-coding-the-future-of-code-or-just-a-short-term-con) and agentic AI functionalities – using it for everything from making evolution simulators to knitting cute jumpers.

![](https://assets.bizclikmedia.net/1800/22a2b8c8fbee36ff6d242a3177d320c3:b7cb7060438e3e472d7bae0807516aa1/1760102618317.webp)

Greg Feingold, Special Projects at Anthropic

Greg Feingold, who works in Special Projects at Anthropic, says on Linkedin that “Cowork is Claude Code for the rest of your work”.

“Since we released Opus 4.5, our agentic coding tool Claude Code has been having a big moment and it's reached beyond the developer world as people discover that it's useful for all sorts of non-coding work: cleaning up your desktop, finding photos on your hard drive, editing spreadsheets, growing plants, knitting sweaters," he adds

“Cowork is our first step towards making Claude Code a useful collaborator for all your non-coding work.

“So far I've used it to mock up a website for a new project proposal, combine PDFs and download images from a Google Doc.

“We shared Cowork today as a research preview for Max subscribers so we can learn how to make it more useful and iterate quickly.”

## How safe is Anthropic Claude Cowork?

By using external connectors, the user can decide the access permissions for the model thereby permitting Claude Cowork to see only the files that they want it to.

Without specifying this explicit access, Claude won’t be able to read or write any files.  

Even when it is given access, it will notify the user before performing significant actions.

Then again, like every new model – especially for something that is in research preview – Claude Cowork also has its risks.

![](https://www.youtube.com/watch?v=WBNZpAWhw5E)

As Claude is capable of performing "potentially destructive action", Anthropic says – such as deleting files – if it thinks it has been instructed to do so, it is imperative that commands given to Claude Cowork must be as clear and precise as possible.

Anthropic also warns about the [risk of prompt injection attacks](https://cybermagazine.com/news/thales-ai-security-fabric-targets-prompt-injection-attacks) from content that Claude might encounter on the internet.

The company says “it has built sophisticated defences against prompt injections, but agent safety – that is, the task of securing Claude’s real-world actions – is still an active area of development in the industry”.

Claude Cowork, although still young, paints an optimistic future where AI agents benefits the wider public.
