# What is Claude Cowork?

## Summary
Nicole Replogle's guide introduces [[claude-cowork]] as [[agentic-ai]] applied to non-technical knowledge workers. Unlike [[claude-code]] (terminal-based for developers), [[claude-cowork]] works with files and folders on your computer plus browser and app integrations, taking autonomous action on your behalf rather than just advising in chat. The tool demonstrates significant advantages over traditional chat interfaces: it can plan work, execute multi-step tasks, ask clarifying questions, and deliver finished files instead of drafted ideas.

## Key Claims
- [[claude-cowork]] is built on the same agentic foundation as [[claude-code]] but targets non-developers
- Feature preview currently macOS-only (Pro or Max subscription); requires desktop app download
- Workflow: user assigns task → Claude develops plan → Claude executes methodically with real-time updates → user reviews finished work
- Access model: click the Cowork tab, select a local folder (sandboxed access), describe task, Claude asks clarifying questions before starting
- Real-time transparency: Claude shows its thinking, reasoning, and each action step taken
- Can be paused to ask permissions for destructive actions (file deletion, etc.); user must click "Allow"
- Integrations via Chrome extension and pre-built connectors: Asana, Notion, ActiveCampaign, HubSpot, Slack, Box, Figma
- [[zapier-mcp]] extends reach to 8,000+ apps via natural language
- Pricing: $20/month Pro or $100/month Max; eats more tokens than chat because it does more work

## Notable Details
- Tested use case: reorganizing business receipts from desktop folder took ~5 minutes, including clarifying questions, sorting, renaming, and generating formatted Excel expense summary
- Can work while user navigates away or focuses on other tabs; desktop app must stay open
- Doesn't retain memory session-to-session (each task starts fresh)
- Not yet available in Projects, on web, Windows, or mobile
- Limitations with Google Workspace apps
- Cannot share chats or artifacts with other users in current preview state
