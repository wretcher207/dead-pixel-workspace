# What is Claude? Everything you need to know about Anthropic's AI powerhouse

## Summary
Comprehensive overview from Tom's Guide covering [[claude]] fundamentals, model lineup, capabilities, and positioning against competitors. [[Anthropic]] distinguishes Claude through [[constitutional-ai]] approach, massive context windows (200k-1M tokens), safety-first design, and recent extensions like [[claude-code]] and [[claude-cowork]]. The guide breaks down which Claude model fits which use case and addresses common questions about cost, safety, and capabilities.

## Key Claims
- Claude is built by Anthropic (founded by former OpenAI researchers) with strong focus on AI alignment and safety
- Constitutional AI uses written ethical principles rather than human thumbs-up/down for training
- Context window up to 200k tokens by default (1M for enterprise), enabling analysis of entire books or codebases
- Extended Thinking Mode allows users to control reasoning depth vs speed trade-off
- Artifact creation enables standalone code, documents, diagrams without chat clutter
- [[claude-cowork]] extends Claude beyond coding to general workflow automation
- Claude Opus 4.6 is frontier model for complex reasoning and agentic workflows
- Claude Sonnet 4.6 balances reasoning and cost for everyday work
- Vision capabilities enable analysis of images, graphs, screenshots, charts
- Pricing uses token-based system; Pro/Max subscriptions cap monthly cost

## Notable Details
- Named after Claude Shannon (father of information theory)
- Differs from ChatGPT by being less chatty, more analytical, more safety-aligned
- Latest model (Claude 3.7 Sonnet) introduces hybrid reasoning in single interface
- Can operate autonomously for extended periods (Opus for ~7 hours)
- Claude 4.5 Sonnet scored 70.3% on SWE-bench verified tasks
- Vision support allows handling of documents, flowcharts, design mockups
- Plugin marketplace is smaller than ChatGPT's but growing
