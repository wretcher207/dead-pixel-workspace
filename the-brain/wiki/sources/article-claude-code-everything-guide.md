# Claude Code Everything You Need to Know

## Summary
A comprehensive GitHub repository-based guide by wesammustafa covering [[claude-code]] mastery from fundamentals through advanced workflows. Spans LLM theory, setup, prompt engineering, slash commands, hooks, automation, integrations, [[mcp]] servers, and the BMAD method. Positioned as the definitive reference for Claude Code proficiency with step-by-step tutorials and real-world examples.

## Key Claims
- Claude Code differs fundamentally from generic AI tools through integrated terminal access and file manipulation
- Prompt engineering requires specific strategies for agentic contexts (not conversational chat)
- Slash commands and hooks provide extensibility without requiring traditional coding
- [[mcp|MCP servers]] enable Claude to interact with external systems and data sources
- The BMAD method (implied from title) offers structured approach to prompt design
- Setup requires clean, intentional configuration to avoid friction
- Hooks execute at lifecycle points: PreToolUse, PostToolUse, Notification, Stop
- Custom commands stored in `.claude/commands/` directory enable reusable shortcuts
- Automation capabilities allow building AI workflows that span tools and platforms
