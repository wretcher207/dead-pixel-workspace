# Claude Cowork: 10 Use Cases I Tested + 67 More by Profession

## Summary

A comprehensive testing report from a professional who spent weeks throwing real work at [[claude-cowork]]. The post covers when to use [[claude-cowork]] vs. regular [[claude]], the five signals indicating a task belongs in [[claude-cowork]] (context window limits, multi-step workflows, direct file manipulation, online research + local data, batch repetition), and honest verdicts on 10 tested use cases: pet photo recognition, article-to-Gumroad conversion (with hallucination warnings), bank reconciliation, content repurposing, and others. The core insight: [[claude-cowork]] excels at autonomous workflows that require file access, extended reasoning, and online integration—not conversation-based tasks.

## Key Claims

- [[claude-cowork]] distinguishes itself through local file access, extended workflows, and integration with multiple data sources
- Five signals indicate a task belongs in [[claude-cowork]]: context limits, multi-step workflows, direct file manipulation, online + local research, batch operations
- Bank reconciliation and expense categorization are near-perfect use cases (extracted correctly, renamed files, flagged discrepancies)
- Content repurposing benefits from existing [[claude-cowork-skills]] (writing styles, performance data analysis)
- Pet photo recognition works but requires caution with edge cases (multiple dogs of same breed)
- Hallucination risk is real when processing similar items (mixing up articles in content conversion)
