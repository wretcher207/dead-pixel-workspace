# If you regularly use Claude for anything at all, what do you use it for?

## Summary

A Reddit discussion exploring diverse use cases for [[claude]], revealing that users employ it far beyond its default conversational mode. Applications range from personal productivity (brainstorming, venting, life planning) to business automation (web scrapers, site builders, rent negotiation), creative work (songwriting, character development), and data visualization. The thread surfaces the tension between [[claude-4.5]] and newer versions, highlights that 8+ hours daily is common for power users, and shows [[claude-code]] enabling non-coders to build functional projects.

## Key Claims

- [[claude]] is effective for brainstorming, external processing, life planning, and routine optimization
- [[claude-code]] enables non-coders to build web scrapers, websites, analytics tools, and automation scripts without prior coding experience
- [[claude-cowork]] excels at summarizing large volumes of information and creating structured outputs (spreadsheets, dashboards)
- Personal productivity applications: weight loss tracking (daily input → coaching + data verification), escape room puzzle automation, travel website building
- Business applications: contract negotiation, insurance optimization, tax strategy, copywriting for Facebook monetization
- Data visualization and presentation work requires minimal AI guidance once the conceptual framework is clear
- Consensus: 15-minute grunt work chunks repeatedly add up; automation of "soul-crushing work" provides measurable quality-of-life improvement
- [[claude-sonnet-4.5]] preferred over [[claude-sonnet-4.6]] for creative writing by some users

## Notable Details

Specific success stories:
- 37-pound weight loss in two months via daily coaching
- $11,000 saved via contract interpretation for escape room business
- $1,000/year auto insurance savings from policy analysis
- Two websites generating "couple thousand dollars" from rent-and-rank model
- Custom RPG for Twitch streaming built over 5 months via iterative Claude guidance
- Fantasy baseball team management: daily roster optimization, injury tracking, free agent recommendations

User profile across sample: 8+ hours/day with Claude is common for power users; costs scale ($200/month for Max plan mentioned as worthwhile).

Limitations noted: image generation poor; time awareness weak; new conversation context reset friction.
