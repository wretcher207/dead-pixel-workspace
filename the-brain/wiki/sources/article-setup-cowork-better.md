# Set Up Claude Cowork Better Than 99% of People

## Summary
Systems Made Better's video guide ("CoWork OS") breaks [[claude-cowork]] setup into seven sequential steps to move users from installation to power usage in one morning. The guide emphasizes installing [[claude-cowork]], writing purposeful global instructions, creating the "About Me" folder, connecting tools via Chrome extension, discovering built-in [[claude-skills]], installing plugins, and practicing until comfortable. The goal is turning [[claude-cowork]] from a confusing feature into a core work tool, with pre-made templates available via CoWork OS (paid product in beta).

## Key Claims
- Seven-step setup framework can be completed in one morning
- [[claude-cowork]] accessible via desktop app download (Pro or Max plan required)
- [[claude-cowork]] tab between Chat and Code; select folder from computer to grant access
- Global instructions are critical: persistent prompts that frame every task
- "About Me" folder contains files that Claude reads before every session (personal context, work style, company goals)
- Built-in [[claude-skills]] available for common workflows (writing, analysis, coding, research, etc.)
- Plugins extend functionality beyond core [[claude-cowork]] (mentioned: Notion, Asana, Slack, etc.)
- Chrome extension connection enables browser automation and web-based task handling
- Sequential steps: (1) Install, (2) Global Instructions, (3) About Me folder, (4) Tool connections, (5) Skills discovery, (6) Plugins, (7) Practice

## Notable Details
- Video chapters: installation (2:15), global instructions (7:58), About Me folder (12:18), tool connections (20:09), skills (32:01), plugins (38:06)

- Emphasis on moving beyond default setup to customized setup
- Integration with Notion, Calendar, Wispr Flow mentioned as part of tool connection
- Practice session recommended as final step (15-20 min)
- Clear mental model: [[claude-cowork]] is not just chat; it's a workspace with persistent context and tool integrations
